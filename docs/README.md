# 业务 Skill 参考文档

业务文档已迁移到 skills/<skill>/references/，由 SkillLoader 按需加载。
该目录仅保留通用说明与模板（如 sql_generator_template.md）。

## 业务 Skill 结构（标准）

```
skills/
	<skill-name>/
		SKILL.md
		scripts/
		references/
```

## 使用方式

- 业务元数据（表名映射/意图与关键词映射）位于 skills/<skill>/references/metadata.md
- 业务规则与示例位于 skills/<skill>/references/
